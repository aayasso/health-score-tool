# Neighborhood Health Score Badge

Embed a live health environment score for any covered ZIP code.

## Embed Snippet

```html
<iframe src="https://hakiksjnpipgstomzzjy.supabase.co/functions/v1/badge?zip=15213" width="280" height="70" frameborder="0" scrolling="no" style="border:none;"></iframe>
```

Replace `15213` with any 5-digit ZIP code.
